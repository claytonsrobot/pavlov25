Pavlov 3D expects your CSV files to be columnar, with a header row. Here is an example of a CSV in the expected format.

When the Basic Model Controls are used, the column name or number can be specified. Also, if you have metadata at the top of your file, and your header row is below this on a different row, you can specify the proper header row by providing a skiprow value.

The skiprow default is 0. The Automatic Model Controls are will use columns 0 and 1 for 2D models. In Excelspeak, columns A and B are columns 0 and 1. Future development is planned for up to seven dimensions. The Autmatic Model Controls will use each successive column for each additional dimension.
