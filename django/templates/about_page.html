Pavlov 3D

Pavlov 3D is an app for converting
columnar CSV data into 3D models,
of FBX filetype.



Using Pavlov 3D, scientists and
STEM students can veiw
collections of data quickly.
Also, users can
extend the shelf life of their data,
easily discuss their data
in a group setting,
and perform high-level analysis.

FBX files can be viewed in
virtual reality (VR) and
augmented reality (AR).
Many features have been added to
leverage the benefits of
CAD Assistant by Open Cascade.

The backend of Pavlov 3D is
compatible with Python 3.9 to 3.12.

The web app version of Pavlov 3D
is available at this URL as of
July 2024.

Initial deployment came during
participation in buildspace n&w s5.
The next deployment target is
the Microsoft Store.

This project started with STL files
generated in MATLAB,
organized and exported using Blender.
